# Lucas S Oliveira
Teste Vaga QA Automação

## Como executar o projeto

Ter o Ruby instalado (versão 2.5 ou Superior) 

### Instalar o bundler
'
gem install bundler
'

### Instalar as depedências do Ruby (projeto)
'
bundle install
'

### Executar localmente
'
bundle exec cucumber
'

### Executar cenário de teste
'
bundle exec cucumber 
ou
bundle exec cucumber -t @tag
'
